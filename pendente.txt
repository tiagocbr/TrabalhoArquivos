--> Criar a funcao bool indices_escreve_registro(FILE* indices,NO no,int RRN_NO)
    Essa função deve receber um NO de acordo com a struct definida em arvore_b.c e escrever no arquivo de indices
    Esse NO deve ser escrito na linha correspondente ao RRN_NO, recebido como parametro
    Se já houver um registro escrito nessa linha, ele sobrescreverá essa linha com o novo NO
    Essa função deve também alterar o cabeçalho do arquivo de indices conforme as especificações do trabalho
--> Criar a funcao  NO criar_no(FILE* indices, int RRN_NO)
    Essa função deve abrir o arquivo de indices,e trazer para a RAM o NO correspondente a linha RRN_NO
    Nao há necessidade de usar malloc nessa função, apenas retorne a variavel NO, sem ponteiros
--> Criar a funcao int get_prox_RRN(FILE* indices) e void set_prox_RRN(FILE* indices, int novo_RRN);
    get_prox_RRN deve abrir o arquivo de indices e retornar o proximo RRN disponivel
    set_prox_RRN seta o proximo RRN disponivel para novo_RRN
